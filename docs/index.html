CLIBX Command-Line Tool Documentation

CLIBX is a command-line tool written in C++ that serves as a package manager for managing C/C++ libraries in your projects.

It allows you to easily install, uninstall, connect, initialize, and manage library dependencies.

Libraries can be installed from Git repositories or as part of kits (library sets). The original prototype was written in Python.

Flags

FlagDescription
--version / -vShows the current CLIBX version.
--help / -hShows this help message and exits.
--force / -fExecutes the command without confirmation.
--web / -wOpens the CLIBX website.
--instDep / -IInstalls dependencies along with the package.
--all / -aConnects all libraries you have installed.

Commands

init

Initializes CLIBX.

clibx init

template

Creates a project template based on the given name.

clibx template <template_name>

install

Installs a library. Supports -f and -I flags.

clibx install <library_name> [-f] [-I]

uninstall

Uninstalls a library. Supports the -f flag.

clibx uninstall <library_name> [-f]

connect

Connects CLIBX to your project. Supports the -a flag.

clibx connect <library_name> [-a]

search

Checks the repository is available

clibx search <library_name>

Installing Kits

Note: Kits are libraries that depend on other libraries within the same set. Their names are usually written in uppercase, and their descriptions specify the included libraries.

To install a kit or library with dependencies:

clibx install "https://github.com/ZeleznaRuda/clibx-mainkit.git" [-f] -I
