CLIBX is a command-line tool created by Rudolf Müller in C++ for managing C/C++ libraries.

Libraries can be installed from Git repositories or as part of kits (library sets). Prototype written in Python.

POSIX only

Compatibility

PlatformCompatibility
LinuxYes
MacOSPartial
WindowsNo

Installation

  • Clone the repository: 
    git clone "https://github.com/rudolfmuller/clibx-package-manager.git"
  • Navigate to the installer: 
    cd clibx-package-manager/compiled/dist/linux/
  • Run the installer (may need permission): 
    chmod +x install.sh && ./install.sh
  • All-in-one line: 
    git clone "https://github.com/rudolfmuller/clibx-package-manager.git" && ./clibx-package-manager/compiled/dist/linux/install.sh

Flags

FlagDescription
--version / -vShow current CLIBX version
--help / -hShow help message
--force / -fExecute command without confirmation
--web / -wOpen CLIBX website
--dep / -dInstall dependencies with package
--all / -aConnect all installed libraries

Commands

  • init – Initializes CLIBX. 
    clibx init
  • template – Creates a project template by name. 
    clibx template <template_name>
  • install – Installs a library. Supports -f and -d. 
    clibx install <library_name> [-f] [-d]
  • uninstall – Uninstalls a library. Supports -f. 
    clibx uninstall <library_name> [-f]
  • connect – Connects CLIBX to your project. Supports -a. 
    clibx connect <library_name> [-a]
  • search – Check if repository is available. 
    clibx search <library_name>
  • ls – List installed libraries. 
    clibx ls
  • info – Show package information. 
    clibx info <library_name>
  • apk-clean – Delete the .clibx folder. Supports -f. 
    clibx clean [-f]
  • apk-uninstall – Uninstall applications. Supports -f. 
    clibx apk-uninstall [-f]

Installing Kits

Note: Kits are sets of libraries that depend on each other. Names are usually uppercase.

  • Install a kit or library with dependencies:

    clibx install <library_name> [-f] -d

Example Usage


    
git clone https://github.com/rudolfmuller/clibx-package-manager.git
cd clibx-package-manager
sudo ./install.sh

mkdir my_project && cd my_project

clibx install "https://github.com/rudolfmuller/platform-lib.git"

echo '#include "libs/platform/platform.h"' > main.cpp
echo 'int main() { platform::getPlatform(); return 0; }' >> main.cpp

clibx connect platform

g++ main.cpp libs/platform/platform.cpp -o main
./main
